## Важные замечания из внутренней документации MWS GPT

### Конфиденциальность данных
Все подключённые модели (`mws/*`) являются **внутренними** — они запускаются на серверах МТС/MWS и данные **не покидают инфраструктуру**. Это важно при работе с чувствительными корпоративными данными. В UI есть признак "Внутренняя/Внешняя", в API этого параметра нет.

### Скорость (TPS — токенов в секунду)
| Модель | TPS (приблизительно) |
|---|---|
| `mws/gpt-oss-120b` | ~2721 tok/s |
| `mws/deepseek-r1-32b` | средняя (длинные цепочки рассуждений) |
| `mws/llama-3.1-8b` | высокая (малый размер) |
| `mws/qwen3-235b` | ниже из-за размера |

> При перегрузке модели запросы уходят в очередь → `APITimeoutError`. Для высоконагруженных сценариев используй streaming и более быстрые модели.

### VLM и русский язык
Внутреннее бенчмаркирование MWS подтверждает: **все открытые VLM значительно слабее на русскоязычных визуальных задачах** (`mm_bench_ru` vs `mm_bench_en`). Дообучение сильно улучшает результат. Именно поэтому `mws/cotype-pro-vl`, обученная на 150K+ русскоязычных корпоративных документов, является первым выбором для русских визуальных задач.

## Детальное описание каждой модели

### CHAT / INSTRUCT

---

#### `mws/gpt-alpha`
- **Upstream:** `mws-gpt-alpha`
- **Тип:** Проприетарная модель MWS (МТС AI)
- **Лучшее:**
  - Универсальный дефолт платформы — сбалансированный выбор для большинства задач
  - Общение, Q&A, суммаризация, деловые тексты на русском и английском
  - Используется как fallback для других моделей в роутере
- **Когда активировать:** нет явных предпочтений по задаче; первый запрос без специфики; корпоративные внутренние задачи

---

#### `mws/qwen2.5-72b`
- **Upstream:** `qwen2.5-72b-instruct` (Alibaba)
- **Параметры:** 72B dense
- **Лучшее:**
  - Надёжная общая модель с сильной поддержкой 29+ языков
  - Coding, math, instruction following на уровне GPT-4o-mini
  - Длинный контекст (128K)
- **Когда активировать:** мультиязычные задачи без необходимости в reasoning-цепочках; как замена gpt-alpha если нужна чуть более мощная модель

---

#### `mws/qwen3-235b`
- **Upstream:** `Qwen3-235B-A22B-Instruct-2507-FP8` (Alibaba)
- **Параметры:** 235B MoE, 22B активных
- **Лучшее:**
  - Флагман серии Qwen3: топ-уровень по reasoning, coding, math
  - Поддерживает thinking mode (extended chain-of-thought) и non-thinking mode в одной модели
  - Конкурент DeepSeek-R1, o1, Gemini-2.5-Pro
  - Поддержка 119 языков, обучен на ~36T токенов
  - Оптимизирован для агентных задач и MCP
- **Когда активировать:** сложный анализ, требующий широкого контекста; когда нужна лучшая открытая модель; fallback для gpt-alpha

---

#### `mws/qwen3-32b`
- **Upstream:** `qwen3-32b` (Alibaba)
- **Параметры:** 32B dense
- **Лучшее:**
  - Сильное рассуждение в компактном размере
  - Thinking mode — пошаговые цепочки для math и logic
  - Быстрее и дешевле чем 235b при сопоставимом качестве на задачах среднего уровня
- **Когда активировать:** задачи требующие reasoning, но без необходимости в максимальной мощи; хороший баланс скорость/качество

---

#### `mws/qwen3-coder`
- **Upstream:** `qwen3-coder-480b-a35b` (Alibaba)
- **Параметры:** 480B MoE, 35B активных
- **Лучшее:**
  - Специализирован для кодирования — лучший выбор для генерации, дебага, рефакторинга кода
  - Поддержка агентного coding (SWE-bench уровень)
  - Fallback: qwen3-235b, gpt-oss-120b
- **Когда активировать:** любые задачи с кодом — написание, объяснение, дебаггинг, код-ревью, генерация тестов

---

#### `mws/llama-3.1-8b`
- **Upstream:** `llama-3.1-8b-instruct` (Meta)
- **Параметры:** 8B dense
- **Лучшее:**
  - Самая быстрая и лёгкая модель в пуле
  - Простые задачи: переформулирование, классификация, короткие ответы, извлечение данных
  - Минимальная латентность, минимальная стоимость
- **Когда активировать:** высоконагруженные сценарии где важна скорость; простые FAQ-боты; pre-processing pipeline

---

#### `mws/llama-3.3-70b`
- **Upstream:** `llama-3.3-70b-instruct` (Meta)
- **Параметры:** 70B dense
- **Лучшее:**
  - Сильная мультиязычная общая модель (Meta SOTA на 70b на момент выпуска)
  - Хорош в instruction following, диалоге, суммаризации
  - Поддержка 8 языков включая русский
  - Fallback для qwen3-235b и gpt-oss-120b
- **Когда активировать:** общая задача без специфики; нужен надёжный мультиязычный backbone; как fallback для крупных моделей

---

#### `mws/gpt-oss-120b`
- **Upstream:** `gpt-oss-120b` (OpenAI, open-weight)
- **Параметры:** 117B, 5.1B активных (MoE-подобная архитектура), MXFP4
- **Лучшее:**
  - Производительность на уровне o4-mini на reasoning бенчмарках
  - Сильные задачи: анализ, рассуждение, следование инструкциям
  - Apache 2.0 лицензия, разработан OpenAI
- **Когда активировать:** когда нужна мощная reasoning-модель от OpenAI без проприетарных ограничений; аналитические задачи enterprise-уровня

---

#### `mws/gpt-oss-20b`
- **Upstream:** `gpt-oss-20b` (OpenAI, open-weight)
- **Параметры:** 21B, 3.6B активных
- **Лучшее:**
  - Аналог o3-mini по производительности
  - Быстрее gpt-oss-120b, может работать на edge-устройствах (16GB)
  - Хорош для локализованных/специализированных сценариев
- **Когда активировать:** нужна качественная модель от OpenAI, но с меньшей латентностью; специализированные дообученные сценарии

---

#### `mws/glm-4.6`
- **Upstream:** `glm-4.6-357b` (Zhipu AI / Z-AI)
- **Параметры:** 357B MoE, ~32B активных
- **Лучшее:**
  - **Кодирование #1** по LiveCodeBench v6 (82.8%), обходит Claude Sonnet 4
  - **Длинный контекст:** 200K input, 128K output — рекорд среди открытых моделей
  - HLE (сложная логика с инструментами): 30.4% vs Claude 4.5's 17.3%
  - Сильный агент: tool use, поисковые агенты
  - MIT лицензия
- **Когда активировать:** сложные задачи кодирования; агентные задачи с инструментами; задачи требующие очень длинного контекста (>128K); аналитика больших документов

---

#### `mws/kimi-k2`
- **Upstream:** `kimi-k2-instruct` (Moonshot AI)
- **Параметры:** 1T MoE, 32B активных
- **Лучшее:**
  - **Лучший агентный выбор:** tool calling, автономное планирование, SWE-bench 65.8%
  - Coding: SOTA среди non-thinking моделей
  - GSM8K: 94.4% (лучше GPT-4.1 и Claude Opus 4)
  - Отлично работает с tool_use API
  - MIT лицензия, open-weight
- **Когда активировать:** агентные задачи с вызовом инструментов; автономные пайплайны; software engineering задачи; когда важна надёжность tool-calling

---

#### `mws/deepseek-r1-32b`
- **Upstream:** `deepseek-r1-distill-qwen-32b` (DeepSeek)
- **Параметры:** 32B dense (дистилляция из R1-671B)
- **Лучшее:**
  - Глубокие structured reasoning цепочки — лучший выбор для math и сложного анализа
  - Подробный "audit trail" рассуждений
  - Выше o1-mini на math/science задачах
- **Когда активировать:** math, физика, логические задачи требующие пошагового решения; задачи где важна трассируемость рассуждений; финансовый и научный анализ

---

#### `mws/qwq-32b`
- **Upstream:** `QwQ-32B` (Alibaba)
- **Параметры:** 32B dense
- **Лучшее:**
  - Быстрое reasoning без длинных цепочек
  - Почти на уровне DeepSeek-R1-671B по качеству, но быстрее
  - Хорош для "черновиков" и повседневных операций с reasoning
- **Когда активировать:** задачи где нужен reasoning, но не нужны сверхдлинные цепочки; быстрое прототипирование; когда deepseek-r1-32b слишком медленный

---

#### `mws/gemma-3-27b`
- **Upstream:** `gemma-3-27b-it` (Google)
- **Параметры:** 27B dense
- **Лучшее:**
  - Мультиязычная (140+ языков), длинный контекст (128K)
  - Хорошее instruction following, безопасность
  - Поддержка мультимодального input (текст + изображение в нативной архитектуре)
  - Открытая Google модель
- **Когда активировать:** мультиязычные задачи где важна безопасность; работа с документами на нескольких языках

---

#### `mws/t-pro`
- **Upstream:** `T-pro-it-1.0` (T-Tech / МТС AI)
- **Параметры:** на базе Qwen 2.5, continual pre-training на 100B+ русских токенов
- **Лучшее:**
  - **Лучший выбор для русского языка:** ruGSM8K 94.1%, ruMATH 77.6%, Arena-Hard-Ru 90.17%
  - Специально обучена для русскоязычных сценариев
  - Русский и английский двуязычная
- **Когда активировать:** задачи целиком или преимущественно на русском языке; русскоязычные чат-боты; работа с русскоязычными документами

### VISION (мультимодальные)

---

#### `mws/qwen3-vl`
- **Upstream:** `qwen3-vl-30b-a3b-instruct` (Alibaba)
- **Параметры:** 30B MoE, 3B активных
- **Лучшее:**
  - Последнее поколение Qwen VL — сильное visual reasoning
  - Анализ графиков, диаграмм, скриншотов, документов
  - Поддержка thinking mode для сложных визуальных задач
- **Когда активировать:** анализ изображений, диаграмм, документов; VQA (visual question answering); задачи с изображениями, требующие рассуждения

---

#### `mws/qwen2.5-vl`
- **Upstream:** `qwen2.5-vl` (Alibaba, базовый)
- **Лучшее:**
  - Лёгкая мультимодальная модель
  - Описание изображений, простые VQA
- **Когда активировать:** простые задачи с изображениями; когда нужна быстрая и недорогая VLM

---

#### `mws/qwen2.5-vl-72b`
- **Upstream:** `qwen2.5-vl-72b` (Alibaba)
- **Параметры:** 72B
- **Лучшее:**
  - Мощная VLM для сложных задач: OCR, таблицы, графики, документы
  - Сильнее базового qwen2.5-vl по точности
- **Когда активировать:** точный OCR; сложные документы с таблицами; детальный анализ изображений

---

#### `mws/cotype-pro-vl`
- **Upstream:** `cotype-pro-vl-32b` (MTS AI)
- **Параметры:** 32B
- **Лучшее:**
  - **Специализирован для русскоязычного enterprise:** финансы, промышленность, IT, телеком, здравоохранение
  - OCR печатных, рукописных и смешанных текстов с учётом визуального контекста
  - 150K+ корпоративных документов в обучении: сканы, скриншоты, договоры, схемы, чертежи
  - Три языка: русский, английский, китайский
  - Перевод с учётом визуального контекста
- **Когда активировать:** российские корпоративные документы; OCR рукописей; анализ скриншотов BI-систем; работа с отраслевыми документами на русском

### EMBEDDINGS

---

#### `mws/bge-m3`
- **Upstream:** `bge-m3` (BAAI)
- **Лучшее:**
  - **Лучший для RAG:** мощные мультиязычные эмбеддинги (100+ языков)
  - Dense + Sparse + ColBERT в одной модели
  - Текущий default для RAG в проекте
- **Когда активировать:** семантический поиск; RAG; кластеризация документов

---

#### `mws/bge-gemma2`
- **Upstream:** `BAAI/bge-multilingual-gemma2` (BAAI)
- **Лучшее:**
  - Мультиязычные эмбеддинги на базе Gemma2
  - Альтернатива bge-m3, может быть точнее на некоторых языках
- **Когда активировать:** альтернативный эмбеддинг; A/B тест качества RAG

---

#### `mws/qwen3-embedding`
- **Upstream:** `qwen3-embedding-8b` (Alibaba)
- **Лучшее:**
  - Новейшие эмбеддинги от Alibaba на базе Qwen3
  - Сильные мультиязычные возможности
- **Когда активировать:** когда нужны современные эмбеддинги от Qwen-экосистемы

---

### STT (Speech-to-Text)

---

#### `mws/whisper-turbo`
- **Upstream:** `whisper-turbo-local` (OpenAI Whisper)
- **Лучшее:** быстрое распознавание речи (default STT в проекте)
- **Когда активировать:** голосовой ввод в реальном времени; всегда первый выбор для STT

#### `mws/whisper-medium`
- **Upstream:** `whisper-medium` (OpenAI Whisper)
- **Лучшее:** более медленный но потенциально точнее для сложных аудио
- **Когда активировать:** когда whisper-turbo даёт ошибки; длинные аудио

---

### IMAGE GENERATION

---

#### `mws/qwen-image`
- **Upstream:** `qwen-image` (Alibaba)
- **Лучшее:** качественная генерация изображений
- **Когда активировать:** генерация изображений по текстовому описанию; основной выбор

#### `mws/qwen-image-lightning`
- **Upstream:** `qwen-image-lightning` (Alibaba)
- **Лучшее:** быстрая генерация изображений (меньше шагов)
- **Когда активировать:** нужна быстрая генерация; превью/черновик изображения

## Матрица выбора для агента

```
Запрос содержит "код"/"программ"/"баг"/"скрипт"  →  mws/qwen3-coder (сложный), mws/kimi-k2 (агентный)
Запрос на русском языке                           →  mws/t-pro (первый выбор)
Запрос содержит математику/логику/задачу          →  mws/deepseek-r1-32b (глубоко), mws/qwq-32b (быстро)
Запрос с изображением (корп. документ RU)         →  mws/cotype-pro-vl
Запрос с изображением (общий)                     →  mws/qwen2.5-vl-72b или mws/qwen3-vl
Запрос содержит "агент"/"инструмент"/"tool"       →  mws/kimi-k2 или mws/glm-4.6
Длинный документ / >100K токенов контекст         →  mws/glm-4.6 (200K)
Поиск/RAG/эмбеддинги                              →  mws/bge-m3
Голосовой ввод                                    →  mws/whisper-turbo
Генерация картинки                                →  mws/qwen-image
Быстрый/дешёвый ответ                            →  mws/llama-3.1-8b
Всё остальное (default)                           →  mws/gpt-alpha → fallback: mws/qwen3-235b
```
